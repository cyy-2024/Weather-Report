Welcome to My Android App

This is a simple Android app designed to demonstrate a music player and schedule manager.

Features

  • Music Player: Play, pause, next, previous functionality.
  • Schedule Manager: Add, edit, and delete reminders.
  • Weather Forecast: Get weather updates using your own API key.

Download and Install

To install the app on your Android device:

  1. Download the APK file from the link below.
  2. On your Android device, go to Settings > Security and enable "Unknown Sources" to allow installation of apps from sources other than the Play Store.
  3. Open the downloaded APK file to install the app.
Download APK

API Key Setup

If you want to use your own weather API key:

  1. Go to the app settings in the app and enter your own API key.
  2. Alternatively, modify the `res/values/strings.xml` or `BuildConfig` to replace the API key in the source code.
